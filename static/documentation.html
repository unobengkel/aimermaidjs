Panduan lengkap operasional Mermaid AI Workspace

Ikuti langkah berikut untuk memulai:

  1. Unduh project dari GitHub atau folder distribusi.
  2. Pastikan komputer Anda telah terinstal Python 3.13 atau versi lebih baru.
  3. Buka Command Prompt (CMD) atau Terminal di direktori project tersebut.
  4. Install seluruh pustaka yang diperlukan dengan perintah: 
    pip install -r requirements.txt
Catatan: Pada proses instalasi pertama kali, beberapa modul mungkin memakan waktu bergantung kecepatan internet Anda.

Project ini menggunakan struktur Client-Server. Pastikan file berikut ada di folder utama:

  • main.py (Server Python)
  • database.py (Database SQLite)
  • rag_handler.py (Logika RAG)
  • static/ (File UI Aplikasi)

Database mermaid_app.db akan terbuat secara otomatis saat aplikasi pertama kali dijalankan.

Via Command Prompt (CMD)

Gunakan perintah berikut untuk menjalankan server:

uvicorn main:app --host 0.0.0.0 --port 8000 --reload

* Opsi --host 0.0.0.0 digunakan agar bisa diakses via IP Local (Tablet/HP).

Via Process Manager

Jika Anda menggunakan PM2 untuk menjalankan secara background:

pm2 start "uvicorn main:app --port 8001" --name mermaid-ai

Sistem membutuhkan API Key dari DeepSeek untuk fungsi chatbot. Key disimpan di LocalStorage browser, aman di sisi pengguna:

  1. Buka halaman **AI Editor**.
  2. Klik tombol Chat (Ungu) di pojok kanan bawah.
  3. Klik ikon Gear/Setelan pada header chat.
  4. Masukkan API Key DeepSeek Anda ke dalam form.
  5. Klik Simpan.
Info: Key ini tetap tersimpan meski browser ditutup, selama Anda tidak membersihkan cache browser.

Interaksi AI

Ketik perintah seperti "Buat diagram flow pendaftaran sekolah dengan 5 tahap" di kolom chat. AI akan membalas dengan kode Mermaid. Klik tombol "Gunakan ke Editor" untuk langsung menampilkan diagram.

Canvas Control

Halaman ini mendukung interaksi penuh:

  • Pan/Geser: Klik dan tarik mouse atau geser dengan 1 jari di tablet.
  • Zoom: Gunakan toolbar (+/-) atau cubit layar dengan 2 jari.
  • Export: Klik "Unduh PNG" untuk menyimpan hasil gambar.

Edit Kembali

Ikon pensil akan memuat ulang diagram tersebut ke dalam Editor utama.

Fitur RAG memungkinkan AI membaca dokumen milik Anda:

  1. Buka halaman **Pustaka Dokumen RAG**.
  2. Unggah file `.pdf` atau `.txt` (Server akan memproses teks menjadi vektor numerik).
  3. Buka **AI Editor**, lalu buka Chat widget.
  4. Klik ikon **Dokumen** pada header Chat.
  5. Pilih dokumen yang ingin dijadikan referensi context.
  6. Tanyakan ke AI, misal: "Buat diagram berdasarkan alur kerja di file PDF yang saya pilih".
