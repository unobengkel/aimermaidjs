Panduan Lengkap Instalasi & Operasional System

1 Download & Instalasi

Ikuti langkah berikut untuk memasang aplikasi di lingkungan lokal Anda:

  1. Ekstrak folder project Mermaid AI ke direktori pilihan Anda.
  2. Pastikan komputer Anda telah terinstal Python 3.13 atau versi lebih baru.
  3. Buka Command Prompt (CMD) atau Terminal di dalam direktori project tersebut.
  4. Jalankan perintah berikut untuk menginstal semua dependensi yang diperlukan:
pip install -r requirements.txt
Penting: Kecepatan instalasi bergantung pada koneksi internet Anda karena sistem akan mengunduh library AI (FastAPI, ChromaDB, HuggingFace).

2 Cara Setup Project

Pastikan struktur file Anda sudah lengkap sebelum menjalankan server:

Backend
  • main.py Server FastAPI
  • database.py Model Data SQLite
  • rag_handler.py Mesin Vector AI
Frontend
  • static/ Folder aset UI (HTML, CSS, JS)
  • requirements.txt Daftar library Python

Database mermaid_app.db dan folder chroma_db akan dibuat otomatis oleh sistem saat pertama kali dijalankan.

3 Menjalankan Aplikasi

A Melalui Command Prompt (CMD)

Gunakan perintah di bawah ini agar aplikasi bisa diakses oleh komputer lain/Tablet dalam satu jaringan:

uvicorn main:app --host 0.0.0.0 --port 8000 --reload

B Melalui Project Manager (PM2 / Background)

Jika ingin menjalankan terus-menerus di latar belakang tanpa harus membuka terminal terus:

pm2 start "uvicorn main:app --port 8000" --name "mermaid-ai"

4 Setting Key AI DeepSeek

Untuk mengaktifkan fitur cerdas AI, Anda memerlukan API Key:

  1. Buka aplikasi lewat browser (localhost:8000).
  2. Klik tombol Chat (Ikon Pesan Ungu) di pojok kanan bawah.
  3. Pilih ikon Roda Gigi (Settings) di bagian atas widget chat.
  4. Tempelkan API Key DeepSeek Anda dan klik Simpan.
Sistem menyimpan Key ini di peramban Anda (Local Storage), bukan di server, sehingga privasi Anda lebih terjaga secara penuh.

5 Halaman AI Mermaid Editor

  • Chatting Perintahkan AI untuk membuat diagram (contoh: "Buat alur login sekolah").
  • Gunakan Kode Klik tombol "Gunakan ke Editor" pada balasan AI untuk langsung memvisualisasikannya.
  • Interaksi Canvas Geser diagram dengan mouse/jari, dan gunakan cubitan (pinch) untuk zoom di tablet/HP.
  • Simpan Beri nama diagram dan klik tombol simpan untuk menyimpannya ke database.

6 Halaman Daftar Diagram

Di halaman ini, Anda dapat mengelola riwayat diagram Anda yang telah tersimpan:

Cari Temukan diagram dengan cepat berdasarkan nama.
Preview Lihat gambar diagram tanpa harus keluar dari daftar.
Edit Buka kembali diagram lama ke halaman editor utama.

7 Pustaka Dukungan RAG

Fitur hebat untuk memberi AI konteks spesifik dari data internal perusahaan/proyek Anda:

  1. Buka halaman Pustaka Dokumen RAG di aplikasi.
  2. Unggah file PDF atau TXT. Sistem akan melakukan "Vektorisasi" dokumen ke database lokal.
  3. Balik ke halaman AI Editor, buka Chat, dan klik ikon Dokumen.
  4. Pilih dokumen Anda sebagai referensi agar jawaban AI 100% sesuai dengan data dokumen tersebut.
